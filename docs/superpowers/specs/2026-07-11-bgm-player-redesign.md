# BGM 播放器色系重制 & 平行四边形改造

## 概述

将 TopBar 左上角 BGM 播放器从红色系矩形改造为棕色+蓝色系平行四边形，右侧无缝衔接屏幕边缘。

## 改动范围

- `src/components/ui/BgmPlayer.tsx` — 色系全部替换、clip-path 平行四边形、边缘细节
- `src/components/layout/TopBar.tsx` — BGM 播放器容器布局调整（右对齐延伸至屏幕边缘）

## 色系映射

| 元素 | 当前 (红) | 目标 (棕+蓝) |
|------|----------|------------|
| 容器背景渐变 | 暗黑 `rgba(20,18,18,0.95)` | 深棕 `#3B2A1E → #2D1F14` |
| 左侧描边 | `rgba(198,47,47,0.5)` 红 | `rgba(100,140,200,0.6)` 蓝 |
| 容器阴影/光晕 | 红色系 | 蓝色系 `rgba(80,130,200,0.08)` |
| 播放按钮 | `#C62F2F → #A51D1D` | `#4A8FD4 → #3570B0` |
| 播放按钮阴影 | 红色 glow | 蓝色 glow `rgba(74,143,212,0.45)` |
| 呼吸光晕 | `rgba(198,47,47,0.4)` | `rgba(74,143,212,0.4)` |
| 唱片中心标 | 红渐变 | 蓝渐变 `#5A9ED8 → #2563A0` |
| 进度条已播放 | 红渐变 | 蓝渐变 `#6AADF0 → #3B7DC0` |
| 循环按钮激活 | `rgba(198,47,47,0.15)` | `rgba(74,143,212,0.15)` |
| 歌名颜色 | `#e8e0d8` | `#e8dcc8` (暖米) |
| 副标题 | `rgba(198,47,47,0.7)` | `rgba(140,170,210,0.7)` |
| 针尖 | `#C62F2F` | `#5A9ED8` |

## 形状

- `clip-path: polygon(12% 0, 100% 0, 100% 100%, 0% 100%)`
- 左侧斜边 ~17° 倾斜，约 30px 缩进
- 右侧完全平直贴合屏幕右边缘

## 边缘细节

- 斜边外层蓝色微光 drop-shadow
- 斜边内侧金属感渐变描边（pseudo-element 或渐变叠加）
- 斜边高光线沿左上→左下

## 布局

- TopBar 中 BGM 容器从 `w-1/3 justify-end` 改为右对齐延伸至屏幕边缘
- 去除右侧多余 padding，确保平行四边形右侧无缝衔接
