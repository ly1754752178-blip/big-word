## 发现的问题

| 严重程度 | 问题 | 说明 |
|---|---|---|
| 中 | 通知重复叠加 | 每次打开 `PersonalStatusOverlay` 都会新增多条通知，多次打开后通知堆叠，遮挡右上角关闭按钮与部分界面 |
| 中 | 通知层遮挡弹窗关闭按钮 | 通知容器 `z-[9999]` 位于固定右上角，与弹窗关闭按钮区域重叠，导致鼠标无法点击关闭按钮（需用 Escape 关闭） |
| 中 | 移动端无法进入个人状态 | `GameLayout` 中左侧边栏与预览面板使用 `hidden lg:flex / hidden lg:block`，在 375px/768px 下 `StatusPreview` 与左侧图标栏均不渲染，用户无法通过正常 UI 打开个人状态 |
| 低 | 375px 顶部音乐控件溢出 | 主界面顶部 banner 中的音乐播放控件在 375px 下超出屏幕右边界 |
| 低 | 当前状态词含「疲惫」 | `StatusPreview` 中当前状态显示为「略显疲惫」，虽未使用「疲劳」字段，但语义上与疲劳相关；属于数据配置问题，非界面实现问题 |
